# VPS/VDS 测评文档库

这是一个专门用于收集和分享云服务器（VPS、VDS）测评的文档仓库。所有测评均以 Markdown 格式撰写，旨在帮助用户做出明智的服务器选择决策。

## 📚 目录结构

```
├── reviews/           # 测评文章目录
│   ├── provider1/     # 按服务商分类
│   ├── provider2/
│   └── ...
├── templates/         # 测评模板
└── docs/             # 相关文档
```

## 🚀 快速开始

### 如何阅读测评

浏览 `reviews/` 目录，选择您感兴趣的服务商或产品进行查看。

### 如何贡献测评

1. 参考 `templates/review-template.md` 模板
2. 在 `reviews/` 目录下对应服务商文件夹中创建新的测评文档
3. 提交 Pull Request

详细信息请参阅 [贡献指南](CONTRIBUTING.md)

## 📋 测评内容

每篇测评通常包含以下内容：

- **基本信息**：服务商、产品型号、价格等
- **硬件配置**：CPU、内存、存储、带宽等
- **性能测试**：CPU性能、磁盘I/O、网络速度等
- **网络测试**：延迟、路由、丢包率等
- **综合评价**：优缺点总结、适用场景等
